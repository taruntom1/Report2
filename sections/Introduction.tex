\section{Introduction}

Vehicle-to-Vehicle (V2V) communication is a key enabler for cooperative safety functions such as collision
warnings, hazard propagation, and context-aware driver assistance. In production-oriented ITS ecosystems,
DSRC (IEEE 802.11p) and cellular V2X (C-V2X) are established technologies intentionally designed for this
domain and continue to be central to large-scale deployments and standardization efforts \cite{secure_iov,lorawan_v2roadside}.
Accordingly, this project does not position LoRa as a replacement for those systems on principle. Instead,
the work is motivated by a practical engineering constraint in our final-year project setting: LoRa hardware
was readily available to us, and it offered a significantly lower-cost platform for building and validating
an end-to-end V2V communication prototype.

Within that constraint, the core technical question addressed in this report is whether a \textit{fully
decentralized} vehicular mesh can provide useful safety-message dissemination without relying on roadside
infrastructure, centralized coordinators, or pre-planned route control. This focus is especially relevant to
academic prototyping and to scenarios where infrastructure support may be sparse, unavailable, or temporarily
disrupted \cite{lora_mesh_emergency,lora_mesh_iot}. The objective, therefore, is not to maximize throughput
against infrastructure-backed systems, but to design and validate a robust distributed communication approach
that remains operational using only peer vehicles and local decision-making.

To realize this objective, the project implements a three-layer architecture: an application layer for safety
message generation and consumption, a network layer for duplicate-aware forwarding and relay control, and a
link layer for radio-facing frame transfer with channel sensing and optional reliability signaling. The
network layer is the primary contribution, as it encapsulates decentralized dissemination logic including
hop-limited propagation, replay suppression, directional eligibility checks, and signal-quality-informed
hold-back forwarding. The same logical core is exercised in both embedded deployment and deterministic
simulation to enable comparable evaluation across real and controlled environments.

This report documents the complete final-year project workflow from design rationale to implementation and
validation. The evaluation combines real-world node trials and deterministic simulation to assess delivery
correctness, forwarding behavior, redundancy suppression, and timing consistency under different operating
conditions. Together, these results provide evidence that a low-cost LoRa-based platform can serve as a
credible foundation for studying decentralized V2V networking principles and cooperative safety dissemination
in an infrastructure-free setting \cite{lora_reliable_v2v_2025,exp_study_lora_vehicle}.
